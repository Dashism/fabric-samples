Welcome to the "Skill Market" app!



This application allows you to complete your profile in order to get in touch with other participants.



You then have two possibilities:


- Receive someone's invitation to participate in a project where your skills will be needed!


- Build your own project to request one or more resources!



The app contains different sections :


Home : you can see a total resume of your profile


Profile : create or update your profile


Project Builder : construct a project with different neeeded skills


Project List : Visualize your old and actual project list


Skills List : access to all the available skills
